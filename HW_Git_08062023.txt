 1.Как отказаться от изменений в файле (файлах) находящийся в рабочем каталоге (work directory)?

git checkout ИМЯ_ФАЙЛА


2.Как вернуть файл из индекса в рабочий каталог?

git restore --staged


3.Как отменить ошибочный commit ?

Удалить  ошибочный , вернувшись на 1 шаг назад
git reset HEAD

git reset HEAD^^     - на 2 шага или так

git reset HEAD~3     - на 3 шага назад


4.Как создать новую ветку и переключится в нее?

git checkout -b ИМЯ_ВЕТКИ


5.В чем отличение между fast-forward merge  and non fast-forfard merge?

Fast-forward merge - слияние, когда ветка, в которую происходит слияние , не имеет новых commit после создания ветки, из которой происходит слияние. Прямое объединение.
Non fast-forfard merge -слияние, когда ветка уже имеет commits после создания ветки, которая будет сливаться. Слияние без прямого перемещения

6.Когда может возникнуть конфликт слияния веток, как  его разрешить?

Когда вносяться изменения из двух веток в один  и тот же файл или часть кода.
Состояние можно проверить командой
git status

Войти в файл в редакторе и оставить или удалить те изменения, которые нужны.
Далее добавить файл и commit
git add
git commit -m ""


7.Можно ли удалить ветку? Как? 

Можно, но при удалении нужно выйти из ветки, которую хотим удалить:

git checkout ДРУГАЯ_ВЕТКА - переключиться на другую ветку
git branch -d ВЕТКА_НА_УДАЛЕНИЕ - удалить ветку
