 1. Как скопировать на свой компьютер репозиторий из github.com, что должно быть сделано предварительно?
1. Проверbть связь локального компьютера с github.com
git config --global -l

если отсуствуют имя и email , то

git config --global user.name  <свое имя>
git config --global user.email <свой email>

2. Проверить наличие ключа SSH в настройках на github. При отсутствии ключа - сгенерировать ключ на локальном компьютере, скопировать в github

3. Создать репозиторий на github
4. Скопировать ссылку на репозиторий в формате SSH
5. На локальном компьютере

git clone <ссылка на репозиторий>
git remote -v



2. Как создать git репозиторий из локального каталога на своем компьютере. Как его отправить на github.com?

Находясь в папке, которую хотим сделать репозиторием

git init
git remote add origin <ссылка на github новый репозиторий>



3.Как получить на локальный компьютер изменения с github.com

git pull

Какую информацию нам дают команда :
git status  - состояние файла в репозитории - добавлен ли он, сделан ли коммит, отправлен ли  github
git log - история commits
git remote -v  - соединение с github локального репозитория
